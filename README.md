# Otterful Otters Website

Official website for Otterful Otters NFT collection.

## Features

- NFT Gallery with search functionality
- Interactive 3D Avatar Builder
- Collection statistics and analytics
- Links to OpenSea and Magic Eden marketplaces
- Responsive design

## Deployment

This site is deployed on Vercel. To deploy:

1. Push to GitHub
2. Import to Vercel
3. Add custom domain: otterfulotters.xyz

## Local Development

Run a local server:
```bash
python server.py
```

Then open http://localhost:8000

## Image Optimization

To create thumbnails for carousels:
```bash
python create_thumbnails.py
```

To optimize images:
```bash
python optimize_images.py
```
